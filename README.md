![Alt text](./img/game.png)

## Dependencies

* Linux
		```
		$ sudo apt-get install libncurses5-dev libncursesw5-dev
		```
* MacOSX
		```
		$ brew install ncurses
		```
* Window
		```
		$ F*ck yourself
		```

## Build
```
$ git clone git@github.com:KennethOng02/Bang.git
$ cd Bang
$ make
```

## Usage
```
$ ./bin/main -p [player_size]
```
or
```
$ make run
```
which defaults to 4 players
